# geo

A time-boxed evaluation of flutter.io, by building out an iOS/Android app.

Using [harmony-ui-kit-sketch-freebie](https://www.sketchappsources.com/free-source/2657-harmony-ui-kit-sketch-freebie-resource.html) as the design spec and a [sanboxed API](https://getsandbox.com/sandboxes/willyweather-stub).

See [Wiki](https://github.com/rhydiant/geo-flutterio/wiki) for more.
